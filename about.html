Jet2

UX/Front End Developer

working in an Agile environments I was heavily involved in developing new and existing re-usable components for the online Jet2holidays - package holidays and Jet2.com - flights only E-commerce web application.

Front-end components were developed in HTML5, SCSS (with BEM approach) and Javascript (Jquery, Vue.js) within the Sitecore CMS. A theming/branding solution was also developed to allow us to personalise Jet2 Holiday types.

CDS

Front End Developer - Metropolitan Police

The Metropolitan Police were in need of a website which allowed users to report crimes and incidents with ease, we developed an efficient online form process for users to find their correct police department.

Episerver was chosen as the preferred CMS and a custom built front-end toolkit was created which comprised of generic tools and helpers to build the templates. The complexity required to build the forms engine and various other UI components needed careful consideration throughout the development process, we adopted a number of techniques with progressive enhancement to optimise the experience for devices of differing types and capabilities.

CDS

Front End Developer - British Army

The British Army campaign was created to encourage young people to join the army at various levels, the month long campaign included several TV and radio appearances.

The microsite consisted of a homepage and a content-hub page which was created responsively using the “mobile-first” approach, the homepage consisted of multiple custom select boxes which narrowed down to the individuals area of interest and included an online application form.

The content-hub page consisted of a video slider with thumbnails and a custom tab for displaying information about the benefits of joining the army, I used custom GA event tracking for all buttons and links and integrated the pages into the OpenText CMS as part of the build process.

For many of the components I used various third party libraries and developed some custom javascript.

CDS

Front End Developer - Bank of England

The Prudential Regulation Authority (PRA) Rulebook website was improved dramatically with an inviting and purposeful design and much needed functionality to the front-end. The custom built CMS allowed the Bank of England editorial and marketing team to create and update content with ease.

A fixed width HMTL layout was used with SASS as the chosen preprocessor. The implementation of Jquery UI used for date pickers, select menus and tabs including custom JavaScript for various page functionality.

Working alongside the backend development team as part of a Waterfall project all the HTML, CSS and JavaScript was integrated into an MVC .NET framework with TFS as the preferred version control.

We tested the website on virtual machines, browsers, real devices and used external tools such as BrowserStack to ensure everything worked as expected.
